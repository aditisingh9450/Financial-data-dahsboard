# Financial Analysis Dashboard - Power BI

## Project Overview

This Power BI dashboard provides a comprehensive financial analysis of transaction data, helping users monitor key business metrics and identify trends through interactive visualizations.

The dashboard enables users to analyze financial performance across different dimensions such as states, customer segments, transaction types, and demographics.

## Key Features

* Dynamic KPI Tracking (Total Amount, Total Transactions, Total Fees, Total Tax)
* Dynamic Metric Selection using Field Parameters
* State-wise Financial Analysis
* Customer Segment Analysis
* Gender-wise Distribution Analysis
* Transaction Type Analysis
* Trend Analysis Over Time
* Interactive Filters and Slicers
* Page Navigation
* Drill-Through Functionality for Detailed Transaction Insights

## Tools & Technologies

* Power BI
* DAX (Data Analysis Expressions)
* Data Modeling
* Field Parameters
* Data Visualization

## Dashboard Insights

The dashboard helps answer business questions such as:

* Which states generate the highest transaction amounts?
* Which customer segments contribute the most revenue?
* How do transaction trends change over time?
* What are the fee and tax distributions across transactions?
* Which transaction types are most frequently used?
